Om mig Kurser Start Fritidsintressen

Dokumentation

För det mesta så gick uppgiften ganska bra! Eftersom jag hade planerat i förväg hur jag skulle göra så stötte
jag inte på några problem av idéslaget, men det betyder tyvärr inte att jag var helt fri från svårigheter...

Här ligger en lista av alla de problem jag stötte på under kodandets gång:

  1. Det första problemet jag stötte på var själva layouten - d.v.s hur jag skulle strukturera sidan. Jag hade idén om att göra en vit "linje" som
    går igenom skärmen, men jag visste inte hur jag skulle implementera det. Jag började med att testa mig fram, men tyvärr så gick det inte så bra.
    Till slut sökte jag på webben och lärde mig lite mer om nav-taggen, och lite om flexbox. Efter mycket trial and error så fick jag till slut till det.
  2. Det andra problemet jag stötte på är själva bubblorna inuti "linjen" som innehåller alla kategorier. Problemet var att jag ville att de skulle
    ta en till de olika sidorna, men jag kunde inte få det att funka, även efter att ha gjort alla .html filer för respektive sida. Detta krävde
    åter igen bara lite trial and error tills jag till slut fick till det. Problemet var att jag helt enkelt glömde lägga till href. Är faktiskt lite generad
    erkänna detta.
  3. Det tredje problemet var att jag ville att sidan du är på ska visas som nedtryckt - d.v.s. att bubblan som leder till sidan har en annan färg
    till skillnad från alla andra. Detta var dock en enkel fix som bara krävde att p-taggen används.
  4. Det femte problemet var att jag hade lite problem med listorna, nämligen ul- och li-taggarna. Problemet var att de inte direkt fungerade som jag kom ihåg
    från FreeCodeCamp. Detta var också en hyfsat enkel fix som krävde lite sökande på webben. Det som är skönt med webbutveckling är att de flesta problem man
    stöter på kan fixas bara om man söker lite på webben!
  5. Det sista och nämligen jobbigaste problemet jag stötte på var att få till text under bilder - captions. Även fast jag hade gjort det på FreeCodeCamp
    så hade jag helt glömt bort hur man gör. Precis som alla andra problem så testade jag att googla en lösning, men jag kunde inte hitta det jag sökte efter.
    Allt jag hittade var antingen för komplicerat eller saker som jag inte hade lärt mig än. Jag testade den klassiska trial and error strategin, men i slutändan så
    fungerade det inte. Efter mycket planlöst letande så gav jag upp och frågade en klasskompis om hjälp. Som tur är fixade han problemet samtidigt som han lärde
    mig hur man gör. Sedan kom problemet med fonten på rubriken, men det löste jag ganska snabbt utan mycket trubbel.

Självklart så kunde jag inte nämna alla problem jag stötte på, eftersom de var ganska okomplicerade och
lätta att fixa. Som sagt så hade jag redan i förväg planerat hur jag skulle göra, men ändå så stötte jag på
många fler problem än vad jag förväntade mig! Att göra en webbsida var inte lika lätt som jag förväntade mig...
